# 中间件管理平台前端

这是一个基于 React + Ant Design 的中间件管理平台前端项目，提供统一的界面来管理和监控各种中间件服务。

## 项目概述

中间件管理平台前端提供了一个现代化的用户界面，集成了以下中间件服务：

- 配置中心
- 注册中心
- 分布式负载均衡
- 流控中心
- 网关中心
- 任务调度
- 消息队列
- 日志平台
- 链路追踪

## 技术栈

- **React**: 前端框架
- **Ant Design**: UI 组件库
- **Vite**: 构建工具
- **CSS**: 样式处理

## 功能特性

### 首页
- 轮播图展示平台特性
- 产品矩阵展示所有中间件服务

### 导航功能
- 右上角"全部"图标展开抽屉导航
- 包含"最近访问"和"所有项目"两个栏目
- 支持环境切换（开发/测试/预发/生产，国内/新加坡）

### 响应式设计
- 支持不同屏幕尺寸的设备
- 用户友好的交互体验

## 快速开始

### 环境要求

- Node.js (版本 16 或更高)
- npm

### 安装和启动

#### 方法一：使用启动脚本（推荐）

```bash
./start.sh
```

#### 方法二：手动启动

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 开发指南

### 组件结构

- `Layout`: 使用 Ant Design 的布局组件
- `Carousel`: 首页轮播图
- `Card`: 产品矩阵卡片组件
- `Drawer`: 抽屉式导航
- `Menu`: 顶部菜单

### 路由管理

使用 URL Hash 实现简单的路由功能，点击模块后在当前页面显示对应内容。

### 图标使用

使用 Ant Design 图标库，为每个中间件服务配置了语义相关的图标。

## 部署

1. 构建生产版本：
```bash
npm run build
```

2. 构建产物位于 `dist/` 目录，可部署到任何静态文件服务器

## 故障排除

### 常见问题

1. **启动失败**: 确保 Node.js 和 npm 已正确安装
2. **依赖安装失败**: 检查网络连接或尝试更换 npm 镜像源
3. **端口占用**: 如果默认端口 3000 被占用，Vite 会自动选择其他端口

### 端口说明

- 开发服务器默认端口：3000
- 如被占用会自动选择其他可用端口（3001, 3002, 等）
